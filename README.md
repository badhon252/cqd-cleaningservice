## âš™ï¸ **Installation & Setup**

### **1. Clone the Repository**

```sh
git clone https://github.com/badhon252/cqd-cleaningservice
cd cqd-cleaningservice
```

### **2. Install Dependencies**

```sh
npm install
```

### **3. Environment Variables**

Create a `.env.local` file and include keys such as:

```
NEXTAUTH_SECRET=
NEXTAUTH_URL=
API_BASE_URL=
GOOGLE_MAPS_API_KEY=
CLOUDINARY_KEY=
CLOUDINARY_SECRET=
```

### **4. Run Development Server**

```sh
npm run dev
```

### **5. Build & Start Production**

```sh
npm run build
npm start
```
